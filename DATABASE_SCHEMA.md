# Startup Incubator AI - Database Schema

## Database Name

```sql
CREATE DATABASE startup_incubator_ai;
USE startup_incubator_ai;
```

---

# Database Tables

## 1. Users Table

Stores user information.

```sql
CREATE TABLE users (
    user_id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    email VARCHAR(150) UNIQUE NOT NULL,
    password VARCHAR(255) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

---

## 2. Startup Ideas Table

Stores startup ideas submitted by users.

```sql
CREATE TABLE startup_ideas (
    idea_id INT AUTO_INCREMENT PRIMARY KEY,
    user_id INT NOT NULL,
    idea_title VARCHAR(255) NOT NULL,
    idea_description TEXT NOT NULL,
    budget_amount DECIMAL(15,2),
    platform VARCHAR(100),
    submitted_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,

    FOREIGN KEY (user_id)
    REFERENCES users(user_id)
    ON DELETE CASCADE
);
```

---

## 3. AI Analysis Table

Stores AI-generated startup analysis results.

```sql
CREATE TABLE ai_analysis (
    analysis_id INT AUTO_INCREMENT PRIMARY KEY,
    idea_id INT NOT NULL,

    success_rate DECIMAL(5,2),
    risk_rate DECIMAL(5,2),
    innovation_score DECIMAL(5,2),

    required_amount DECIMAL(15,2),
    budget_status VARCHAR(50),

    analysis_date TIMESTAMP DEFAULT CURRENT_TIMESTAMP,

    FOREIGN KEY (idea_id)
    REFERENCES startup_ideas(idea_id)
    ON DELETE CASCADE
);
```

---

## 4. Recommendations Table

Stores AI recommendations.

```sql
CREATE TABLE recommendations (
    recommendation_id INT AUTO_INCREMENT PRIMARY KEY,
    analysis_id INT NOT NULL,
    recommendation_text TEXT NOT NULL,

    FOREIGN KEY (analysis_id)
    REFERENCES ai_analysis(analysis_id)
    ON DELETE CASCADE
);
```

---

## 5. Implementation Steps Table

Stores startup implementation roadmap.

```sql
CREATE TABLE implementation_steps (
    step_id INT AUTO_INCREMENT PRIMARY KEY,
    analysis_id INT NOT NULL,

    step_number INT,
    step_description TEXT,

    FOREIGN KEY (analysis_id)
    REFERENCES ai_analysis(analysis_id)
    ON DELETE CASCADE
);
```

---

## 6. Competitor Analysis Table

Stores competitor details.

```sql
CREATE TABLE competitor_analysis (
    competitor_id INT AUTO_INCREMENT PRIMARY KEY,
    analysis_id INT NOT NULL,

    competitor_name VARCHAR(255),
    strengths TEXT,
    weaknesses TEXT,
    market_share VARCHAR(50),

    FOREIGN KEY (analysis_id)
    REFERENCES ai_analysis(analysis_id)
    ON DELETE CASCADE
);
```

---

## 7. Similar Startups Table

Stores startup examples generated by AI.

```sql
CREATE TABLE similar_startups (
    startup_id INT AUTO_INCREMENT PRIMARY KEY,
    analysis_id INT NOT NULL,

    startup_name VARCHAR(255),
    success_percentage DECIMAL(5,2),
    failure_percentage DECIMAL(5,2),
    country VARCHAR(100),

    FOREIGN KEY (analysis_id)
    REFERENCES ai_analysis(analysis_id)
    ON DELETE CASCADE
);
```

---

## 8. PDF Reports Table

Stores generated PDF reports.

```sql
CREATE TABLE pdf_reports (
    report_id INT AUTO_INCREMENT PRIMARY KEY,
    analysis_id INT NOT NULL,

    file_name VARCHAR(255),
    file_path VARCHAR(500),
    generated_date TIMESTAMP DEFAULT CURRENT_TIMESTAMP,

    FOREIGN KEY (analysis_id)
    REFERENCES ai_analysis(analysis_id)
    ON DELETE CASCADE
);
```
